# 各Skillの説明

## frontend-design（Anthropic公式）

- **目的**  
  見た目がありきたりではない、**高品質で印象に残るフロントエンド**を作るためのSkill

- **やること**  
  まず、画面の目的・対象ユーザー・デザインの方向性を決める

- **重要な考え方**  
  無難なAIっぽいデザインではなく、**はっきりした世界観**を持たせる

- **重視する点**  
  - タイポグラフィ
  - 色使い
  - アニメーション
  - レイアウト
  - 背景や装飾の作り込み

- **禁止されていること**  
  - よくある量産型UI
  - ありがちな配色
  - 無個性なフォント
  - テンプレ感の強いデザイン

- **最終的に求めるもの**  
  **デザイン性が高く、実際に動く本番品質のUI実装**
