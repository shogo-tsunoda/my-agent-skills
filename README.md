# my-agent-skills

Claude CodeやCodexで自分用に使えるスキルを集めたリポジトリです。

## ディレクトリ構成

| ディレクトリ | 内容 |
|---|---|
| `skills/claude-code/` | Claude Code 専用スキル |
| `skills/web/` | Web フロントエンド向けスキル |
| `skills/react/` | React / Next.js 向けスキル |
| `skills/document/` | ドキュメント・デザイン向けスキル |
| `skills/react-native/` | React Native / Expo 向けスキル |
