# ドキュメント・デザイン スキル

ドキュメント作成やスタイリングに関するスキルです。技術スタック非依存で使えます。

**出所**: [anthropics/skills](https://github.com/anthropics/skills)

---

## doc-coauthoring

ドキュメントを AI と段階的に共同作成するためのスキルです。

- **用途**: 提案書・設計書・仕様書・PRD などの構造化ドキュメント作成
- **流れ**: コンテキスト収集 → 構成・執筆 → 読者目線でのレビュー
- **ポイント**: いきなり全文を書かず、段階的に情報を整理しながら書き上げる
- **場所**: `skills/document/doc-coauthoring/`

---

## brand-guidelines

Anthropic の公式ブランドカラーとタイポグラフィをアーティファクトに適用するスキルです。

- **用途**: Anthropic ブランドの見た目・スタイルを適用したいとき
- **内容**: 公式カラーパレット・フォント・ビジュアルデザインの基準
- **場所**: `skills/document/brand-guidelines/`

---

## theme-factory

スライド・ドキュメント・HTML ページなど様々なアーティファクトにテーマを適用するスキルです。

- **用途**: アーティファクトの見た目を統一・整えたいとき
- **内容**: 10 種類のプリセットテーマ（配色 + フォントペア）を収録
- **テーマ例**: arctic-frost / midnight-galaxy / golden-hour / ocean-depths など
- **場所**: `skills/document/theme-factory/`
